# 个人作品集网站 PRD

## 项目概述
基于React + TypeScript + Tailwind CSS + Framer Motion构建的现代化个人作品集网站，采用深色主题设计，具备流畅的动画效果和响应式布局。

## 核心功能

### 1. 首页 (Home)
- **大标题展示**：个人姓名突出显示
- **动态打字效果**：个人简介使用打字机动画效果
- **头像展示**：圆形头像，支持动画效果
- **视口触发动画**：组件进入视口时触发入场动画
- **平滑滚动导航**：支持页面间平滑滚动

### 2. 关于我 (About)
- **个人照片展示**：双照片布局，支持左右摆动动画
- **个人描述**：详细的个人介绍，支持打字机动画
- **详细信息卡片**：工作经验、个人偶像、个人荣誉、职业目标等分类展示
- **照片动画效果**：两张照片具有相反方向的摆动动画，增强视觉层次感
- **布局优化**："关于我"和个人描述位于组件布局上方

### 3. 项目展示 (Projects)
- **多卡片轮播布局**：同时显示3张项目卡片
- **滑动切换功能**：支持左右箭头按钮切换和键盘导航
- **项目卡片组件**：包含项目标题、截图、描述、技术栈、预览链接和详情链接
- **动画效果**：
  - 卡片滑入滑出动画（新卡片从右侧滑入，旧卡片向左侧滑出）
  - 弹簧物理动画提供真实感
  - 卡片缩放和透明度变化
  - 3D旋转效果增强立体感
- **交互功能**：
  - 鼠标悬停显示项目描述和技术栈
  - 预览和查看图标居中显示，支持缩放效果
  - 圆点指示器快速跳转
  - 当前页码显示

### 4. 联系方式 (Contact)
- **水平卡片布局**：联系方式以水平卡片形式展示
- **图标展示**：使用emoji图标直观展示联系方式类型
- **二维码预览**：微信和支付宝支持悬停显示二维码
  - 二维码尺寸：128px × 128px，确保清晰可扫描
  - 预览框优化：适当的内边距和最小宽度
- **社交媒体链接**：LinkedIn、Twitter、Instagram、Blog等社交媒体平台
- **回到顶部功能**：页面底部提供回到顶部按钮

## 技术实现

### 前端技术栈
- **框架**：React 18.2.0 + TypeScript
- **样式**：Tailwind CSS 3.3.6
- **动画**：Framer Motion 10.16.16
- **构建工具**：Vite 5.0.8
- **代码质量**：ESLint + TypeScript类型检查

### 项目结构
```
src/
├── components/          # 可复用组件
│   └── ui/             # UI组件
│       └── TypingAnimation.tsx
├── data/               # 数据层
│   └── personalData.ts # 个人信息配置
├── sections/           # 页面区块
│   ├── Home/           # 首页
│   ├── About/          # 关于我
│   ├── Projects/       # 项目展示
│   └── Contact/        # 联系方式
├── styles/             # 样式文件
│   └── globals.css
└── App.tsx            # 应用入口
```

### 数据管理
- **集中式数据配置**：所有个人信息、项目数据、联系方式统一在`personalData.ts`中管理
- **类型安全**：使用TypeScript接口定义数据结构
- **易于维护**：数据与组件分离，方便后续修改

## 设计要求

### 视觉设计
- **简洁现代的设计风格**：采用扁平化设计理念
- **深色主题**：主色调为深色系，搭配高对比度的强调色
- **流畅的滚动动画**：使用Framer Motion实现平滑的页面过渡
- **移动端适配**：响应式设计，支持各种屏幕尺寸

### 动画效果
- **视口触发动画**：组件进入视口时自动触发动画
- **层次化动画**：不同元素使用不同的动画延迟和持续时间
- **物理真实感**：弹簧动画提供自然的物理反馈
- **交互反馈**：悬停、点击等交互提供即时视觉反馈

### 用户体验
- **直观的导航**：清晰的页面结构和导航指示
- **快速加载**：使用Vite构建工具优化加载性能
- **可访问性**：支持键盘导航和屏幕阅读器
- **跨浏览器兼容**：确保主流浏览器的兼容性
